# Sphere10.Framework.Data.Oracle

**Oracle implementation** for Sphere10.Framework.Data abstraction layer, enabling database access using Oracle.ManagedDataAccess.Core, the official Oracle .NET data provider.

Compatible with Oracle 12c+ and Oracle Autonomous Database.

## Installation

```bash
dotnet add package Sphere10.Framework.Data.Oracle
```
